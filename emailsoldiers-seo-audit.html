03 · Технические проблемы

Топ-7 блокеров

🔴 PHP-ошибка в robots.txt
Критично #1

robots.txt возвращает PHP Warning в теле ответа:
Warning: fopen(/var/www/.../wp-content/sitemaps/174564472969f3679e69b0b): failed to open stream: No such file or directory
Плагин xml-sitemaps.php пытается открыть временный файл, которого нет. Ошибка выводится прямо в robots.txt — поисковые боты видят мусор после директив. Нужно либо переустановить/обновить плагин xml-sitemaps, либо пересоздать sitemap вручную.

🔴 nginx редиректы с :443 + www не закрыт
Критично #2

HTTP→HTTPS редирект отдаёт порт в Location:
http://emailsoldiers.ru/blog → https://emailsoldiers.ru:443/blog
Дополнительно: www.emailsoldiers.ru НЕ редиректит на non-www — отдаёт https://www.emailsoldiers.ru:443/. Два хоста в индексе = дубли, размытие ссылочного веса. Фикс — в nginx.conf, нужен сисадмин.

🔴 /services/ → 404
Критично #3

Стандартный URL для страницы услуг (/services/) возвращает 404. Реальная страница — на нестандартном /uslugi. Любые внешние ссылки на /services/ теряются. Рекомендация: 301-редирект /services/ → /uslugi (или переименовать slug).

🟠 Массовые пустые alt-тексты
Средне #4

Главная: 48/55 изображений без alt (87%). About-us: 264/264 (100%). Blog: 47/61. Glossary: 17/17. Cases: 15/15. Масштаб проблемы огромный — несколько тысяч изображений по всему сайту.

🟠 H1 отсутствует на /about-us
Средне #5

Страница «О нас» (/about-us) не содержит ни одного H1-заголовка. При этом страница содержит 264 изображения без alt — вероятно, это слайдовая презентация, загруженная как картинки.

🟠 Meta description отсутствует на /glossary/ и других
Средне #6

Страница-хаб глоссария (/glossary/) не имеет meta description (длина 0). Аналогично — все отдельные статьи глоссария нужно проверить на наличие описаний.

⚪ H1 /blog/ — «Все статьи»
Мелочь #7

H1 страницы блога — «Все статьи». Не содержит ключевых слов, не помогает ранжированию. Лучше: «Блог о CRM- и email-маркетинге» или «Статьи об email-рассылках и CRM».

10 · SWOT

Сильные стороны и риски

Сильные стороны
  • Большая контентная база: 306 постов, 91 глоссарий, 16 кейсов
  • 8+ лет опыта и 200+ клиентов — есть что показать
  • Глоссарий работает — топ-страницы по информационным запросам
  • Активный блог (последнее обновление: март 2026)
Слабые стороны
  • DR=13 — очень низкий для 8-летнего агентства
  • PHP-ошибка в robots.txt — базовая техника не работает
  • www-дубль не закрыт — дробит ссылочный вес
  • Ни одной коммерческой страницы в топ-10
  • About-us без H1 и с 264 изображениями без alt
Возможности
  • Глоссарий как точка входа → перелинковка на услуги
  • Ниша «структура email-письма» (263k спрос) — у конкурентов слабо закрыта
  • DR низкий — есть пространство для быстрого роста линкбилдингом
  • Кейсы с реальными клиентами (Додо, Miele, Vichy) — контент-маркетинг
Риски
  • Февральское падение VIS −65% — причина неизвестна. Возможны санкции Яндекса
  • emailmatrix.ru активно заполняет информационные ниши
  • PHP-ошибка robots.txt — если боты не могут читать директивы, риск переиндексации мусора
