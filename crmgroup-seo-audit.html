Ключевые выводы

За 10 лет работы агентство не превратило сайт в канал лидогенерации. VIS=115 — это уровень молодого стартапа, не состоявшегося агентства с 400+ проектами. Проблема не в конкуренции — в техническом и контентном долге.

  • /email-marketing-pod-klyuch/ сломана — нет meta description (NULL) + двойной H1. Google не показывает сниппет, CTR = 0
  • Сервисные страницы не ранжируются — ни /crm-marketing/, ни /email-marketing-pod-klyuch/ не попали в топ-10 по органике. Весь трафик — с глоссария
  • 32 из 38 изображений без alt на /special-projects/. Яндекс активно использует image search
  • 39 страниц категорий/тегов в sitemap без noindex — тратят crawl budget при 49% индексации
🟢 Точки быстрого роста
  • Программа лояльности — WS=73 581, в топе только SaaS-платформы. Нет ни одного CRM-агентства. Голубой океан
  • Структура email-письма — emailmatrix.ru владеет кластером WS=260 000. Статья с кейсами Miele/BK перехватит трафик
  • FAQPage JSON-LD на /crm-marketing/ и /email-marketing-pod-klyuch/ — FAQ-блоки уже есть, нужна только разметка → rich snippets через 2–4 недели
  • Фикс /email-marketing-pod-klyuch/ — добавить meta description (155 символов) + убрать второй H1. ≤ 1 часа работы

03 · Критические проблемы

Топ-5 блокеров роста

Технические фиксы устраняются до контентных работ — иначе новые страницы недополучат 30–50% потенциала.

1

Главная коммерческая страница технически сломана

/email-marketing-pod-klyuch/ — ключевая страница под запрос «email маркетинг под ключ» (WS ~2 000) не имеет meta description и содержит два H1-заголовка. Google не отображает сниппет, CTR = 0. Inbox-marketing.ru уже в топ-1 по этому запросу.

2

VIS=115 при 10-летней истории агентства — 3× разрыв с конкурентом

emailmatrix.ru (аналогичный прямой конкурент) имеет VIS=214, DR=18, топ-50=7 389 ключей против 2 456 у crmgroup.ru. Разрыв в 3× по семантическому охвату. Сайт фактически не работает как канал лидогенерации — 90%+ трафика идёт с брендовых и информационных запросов.

Сервисные страницы не генерируют органический трафик

Ни /crm-marketing/, ни /services/, ни /marketing-audit/ не попали в топ-10 страниц по органическому трафику. Весь органический трафик идёт с глоссария и блога (информационные ключи). Агентство присутствует в поиске только как образовательный ресурс, не как поставщик услуг.

Массовое отсутствие alt-текста на изображениях

/special-projects/ — 32 из 38 изображений без alt (84%). /web-development/ — 10 из 17, /marketing-audit/ — 7 из 12. Яндекс активно использует image search и текстовую релевантность изображений для оценки страниц. Потеря ранжирования в Яндекс.Картинках и ухудшение сигнала релевантности.

07 · Быстрые победы

5 действий на этой неделе

Все 5 пунктов — ≤ 1 рабочего дня суммарно. Каждый из них напрямую влияет на индексацию или CTR в поиске.

Meta description для /email-marketing-pod-klyuch/
Добавить 155 символов в Yoast SEO. Сейчас Google не показывает описание — пустой сниппет = CTR 0%.
≤ 1ч crit
Убрать второй H1 на /email-marketing-pod-klyuch/
В коде страницы явно два H1-тега. Дублирование нарушает структуру документа и снижает понятность для поисковика.
≤ 30м crit
FAQPage JSON-LD на /crm-marketing/ и /email-marketing-pod-klyuch/
FAQ-блоки уже присутствуют на обеих страницах. Нужна только JSON-LD разметка. Rich snippets в Яндексе через 2–4 недели.
2–3ч high
Noindex для /category/* и /tag/* страниц блога
39 страниц-архивов тратят crawl budget. Один toggle в Yoast SEO → Поисковые системы → Архивы. Освобождает бот для полезных страниц.
≤ 15м med
OG type → website на /crm-marketing/
Сейчас og:type = "article". При репосте в мессенджерах карточка выглядит как статья. Изменить в Yoast SEO → Social → Open Graph.
≤ 15м med
08 · Роадмап

План на 3 месяца

Технические фиксы → контентные быстрые победы → развитие охвата. Последовательность важна: без технического фундамента контент недополучает видимость.

Этап 1
Месяц 1 — Тех. фундамент
  • Meta description + H1 фикс /email-marketing-pod-klyuch/
  • FAQPage JSON-LD на 2 коммерческих страницах
  • OG type → website на /crm-marketing/
  • Noindex category/tag страниц блога (39 шт)
  • Alt-тексты массово (special-projects, web-dev, audit)
  • H1 /services/ переписать с ключом
  • Nginx redirect: убрать :443
  • Закрыть testcontent.crmgroup.ru
  • Убрать конфликт /conditions/ в robots/sitemap
Этап 2
Месяц 1–2 — Контент
  • Статья «Структура email-письма с примерами» (2 500–3 000 слов, /blog/email-structure/)
  • Статья «Программа лояльности: виды и как запустить» (2 500 слов)
  • Страница-услуга /loyalty-program/ под транзакционный спрос
  • Переработка /crm-marketing/ с кластером «crm агентство»
  • Кейсы с метриками по loyalty на портфолио-странице
Этап 3
Месяц 2–3 — Охват
  • Статья «Push-уведомления: полное руководство» (2 000 слов)
  • Статья «Брошенная корзина: письма с примерами»
  • /services/ как hub-страница с перелинковкой на все сервисы
  • Линкбилдинг: гостевые публикации Cossa, vc.ru, Habr
  • Подключить GSC + Яндекс.Вебмастер
  • Статья «UTM метки: полный гайд» (после укрепления DR)

10 · SWOT-анализ

Сильные стороны и риски

SEO-позиция агентства с точки зрения органического продвижения.

Сильные стороны
  • 10 лет работы, 400+ проектов — авторитет в нише
  • Кейсы с крупными брендами (Miele, Burger King, Kérastase)
  • Глоссарий из 103 страниц — SEO-актив с трафиком
  • WordPress + Yoast SEO — быстрые технические правки
  • Продуктовая экспертиза по loyalty и email-маркетингу
Слабые стороны
  • DR=16 — ссылочный авторитет не соответствует возрасту
  • Сломана ключевая коммерческая страница (null meta desc)
  • 0 коммерческих запросов в топ-10 — агентство «невидимо» для клиента
  • 49% страниц не проиндексировано Google (134/276)
  • H1 главной страницы — слоган без ключей
Возможности
  • «Программа лояльности» WS=73k — нет агентства в топе, голубой океан
  • «Структура email-письма» WS=260k — перехват у emailmatrix с кейсами
  • FAQPage JSON-LD → rich snippets → рост CTR за 2–4 недели
  • Страница /loyalty-program/ — первый агентский игрок в нише
  • Линкбилдинг через Cossa/vc.ru поднимет DR быстро (низкая база)
Угрозы
  • emailmatrix.ru DR=18 ускоряется — разрыв в семантике растёт
  • mindbox.ru DR=40+ захватывает образовательный контент ниши
  • Inbox-marketing уже в топ-1 по «email маркетинг под ключ» при DR=14
  • Low crawl budget (49% индекс) — новые страницы медленно индексируются
  • Тестовый субдомен в индексе — риск утечки контента
