01 — Executive Summary

Топ-5 критичных проблем

Сайт имеет 461 страницу в Яндекс-индексе, VIS=112 — но 95%+ трафика брендовый. Небрендового органического трафика почти нет. Конкурент с почти одинаковым DR получает в 2.8× больше видимости за счёт контентной стратегии.

1

95%+ трафика брендовый — небрендовое SEO не работает

VIS=112, но почти весь трафик — запросы «aim clo», «аим кло», «aim clo одежда». Небрендовый органический трафик — менее 10 vis/мес. Категорийные страницы (/shop/platya/, /shop/svitery/) ранжируются на pos 24–43 по высокочастотным запросам — фактически вне видимой выдачи. Конкурент uniquefabric.ru (DR=25, почти равный DR=24) получает VIS=310 — в 2.8× больше — за счёт страниц материалов и SEO-подкатегорий.

2

Нет страниц материалов — главный gap vs uniquefabric

«Натуральные ткани» — это USP бренда (меринос, кашемир, лён, хлопок). Uniquefabric.ru захватил весь кластер «модал ткань» (WS=7K–12K) через /materials/modal/. У aimclo нет ни одной страницы /materials/*. Упущено 15+ ключей с суммарным WS ~70K — прямые запросы целевой аудитории.

3

Категории без SEO-подразделов — Google и Яндекс видят один URL

/shop/platya/ — одна страница без подразделов по типу (серое, свободное, с декольте, кружевное). Uniquefabric.ru имеет /platya/serye, /platya/svobodnogo-kroya, /platya/s-dekolte — каждая в top-10 по своему кластеру. У aimclo — одна страница, собирающая мусорный трафик на pos 30–43.

5

HTML 291–494 KB, отсутствие H1 на главной — технические тормоза

HTML страниц 291–494 KB при норме 50–150 KB — в 3–5 раз тяжелее. Главная страница не имеет H1: вместо него H2 со слоганом «Ты — художник своего мира» (не ключевое слово). Поисковик не получает сигнал о теме страницы. Правка H1 занимает 5 минут в Bitrix.

08 — Quick Wins

5 правок за 1 рабочий день

Эти правки не требуют разработки или написания контента. Каждая выполняется за 5–30 минут. Яндекс видит 461 страницу — технический барьер снят, теперь фокус на качестве страниц.

1. H1 на главной странице
В Bitrix: открыть редактор главной -> добавить выше H2 тег H1 с ключом «Женская одежда из натуральных тканей». Яндекс получит сигнал о теме страницы.
до 10 минвысокий
2. Оптимизировать /blog/ — title + H1 + meta description
Title: «Блог Aim Clo — о натуральных тканях, стиле и базовом гардеробе» (60 симв.). H1 + meta description 150 симв.
до 20 минвысокий
3. Добавить sitemap-blog.xml
Собрать все URL /blog/*, создать отдельный sitemap, добавить в siteindex. В Bitrix можно настроить автогенерацию. Нужно для Google — Яндекс уже индексирует 461 страницу.
до 1 часавысокий
4. Исправить http -> https в robots.txt (секция Yandex)
Найти строку Sitemap: http://aimclo.ru/sitemap.xml -> заменить на https://. Одна строка в файле.
до 5 минсредний
5. Уникальный meta description для топ-15 категорий
Сейчас 265–289 симв. на всех страницах — шаблон. Написать 150–160 симв. с ключом и CTA для каждой категории. Bitrix позволяет задать в свойствах раздела.
до 2 часовсредний
