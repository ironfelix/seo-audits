01 — Executive Summary

Топ-5 критичных проблем

Сайт фактически не существует в органическом поиске. При 1 120 страницах в sitemap — 0 проиндексировано в Яндексе, около 10 в Google. Все коммерческие страницы категорий и карточек вне выдачи.

1

Яндекс: 0 страниц в индексе

При 1 120 страницах в sitemap — ни одна не проиндексирована Яндексом. Целевой поисковик (из брифа) не видит сайт. Органический трафик из Яндекса = 0. Вероятная причина: сайт не подавался в Яндекс.Вебмастер, Яндекс не получил сигнал о существовании сайта.

site:aimclo.ru — 0 результатов (Яндекс, апрель 2026)
Критично
2

Google: ~10 страниц из 1 120+

Все товарные категории и карточки товаров вне Google. Проиндексированы только: главная, 5 категорий верхнего уровня, FAQ, шоурум, акции. Карточки и блог — полностью вне выдачи. Причины: тяжёлые страницы 291–494 KB, статьи блога не в sitemap, параметрические дубли тратят crawl budget.

site:aimclo.ru — ~10 результатов (Google, апрель 2026)
Критично
3

Нет H1 на главной странице

Главная — точка входа по брендовым запросам. Вместо H1 стоит H2 со слоганом «Ты — художник своего мира» (не ключевое слово). Поисковик не получает сигнал о теме страницы. Правка занимает 5 минут в Bitrix.

h1_count=0 на / (crawl.py, 27.04.2026)
Высокий
4

Раздел /blog/ полностью не оптимизирован

Title «Блог» (4 символа). Нет H1, нет meta description. Около 20 статей блога не добавлены в sitemap.xml — краулеры их не находят. Весь информационный трафик заблокирован на уровне точки входа.

/blog/ title=«Блог» (4 симв.), h1=null, meta_description=null
Высокий
5

0 страниц под ключевое USP: натуральные ткани

aimclo позиционируется как «одежда из натуральных материалов» (меринос, кашемир, лён, хлопок). Но нет ни одной SEO-страницы под эти запросы. uniquefabric.ru (DR=25, почти равен aimclo DR=24) получает в 2.7× больше vis (315 vs 117) именно за счёт страниц материалов /materials/*. bonnywool.ru держит топ-5 по меринос-запросам с DR=13 и почти без текста — рынок совершенно не занят.

Ключевой инсайт: uniquefabric.ru как benchmark

uniquefabric.ru при DR=25 (почти равном aimclo DR=24) получает vis=315 против vis=117 у aimclo.ru — в 2.7 раза больше при одинаковом авторитете домена. Разница — в контентной стратегии: uniquefabric создаёт страницы под материалы (/materials/tensel, /materials/modal, /materials/kupra), каждая приносит 78–206 ключей в топ-50. aimclo.ru этого не делает совсем.

07 — Quick Wins

5 правок за 1 рабочий день

Эти правки не требуют разработки или написания контента. Каждая выполняется за 5–30 минут.

1. Яндекс.Вебмастер — зарегистрировать и верифицировать
Создать аккаунт -> добавить сайт -> добавить sitemap.xml. Без этого шага все действия для Яндекса = ноль.
до 30 минкритично
2. H1 на главной странице
В Bitrix: открыть редактор главной -> добавить выше H2 тег H1 с ключом «Женская одежда из натуральных тканей».
до 10 минвысокий
3. Оптимизировать /blog/ — title + H1 + meta description
Title: «Блог Aim Clo — о натуральных тканях, стиле и базовом гардеробе» (60 симв.). H1 + meta description 150 симв.
до 20 минвысокий
4. Добавить sitemap-blog.xml
Собрать все URL /blog/*, создать отдельный sitemap, добавить в siteindex. В Bitrix можно настроить автогенерацию.
до 1 часавысокий
5. Исправить http -> https в robots.txt (секция Yandex)
Найти строку Sitemap: http://aimclo.ru/sitemap.xml -> заменить на https://. Одна строка в файле.
до 5 минсредний

10 — SWOT

Сильные стороны и риски

По результатам трёх блоков аудита: технический, контентный, конкурентный.

Strengths — Сильные стороны
  • DR=24 — выше всех прямых нишевых конкурентов (bonnywool DR=13, tobeone DR=21)
  • 1 120+ страниц в sitemap — объём каталога есть
  • SSL, редиректы, robots.txt — всё работает корректно
  • Чёткое USP: натуральные материалы (меринос, кашемир, лён)
  • Шоурум в Новосибирске — локальный сигнал для поисковиков
  • Статьи блога существуют (~20) — контент есть, нужна оптимизация
Weaknesses — Слабые стороны
  • Яндекс: 0 страниц в индексе — корневая проблема
  • 95%+ видимости — брендовый трафик, не коммерческий
  • HTML 291–494 KB — 3–5 раз тяжелее нормы
  • Нет страниц под ключевое USP (меринос, кашемир, лён)
  • Нет H1 на главной, /blog/ без SEO
  • Шаблонный meta description на всех страницах
Opportunities — Возможности
  • Рынок не занят: bonnywool в топ с DR=13 без текста
  • 3 материальных страницы = +50–100% vis за 2–4 месяца
  • Базовый гардероб: уже ~pos 10, малые усилия для топ-5
  • uniquefabric не имеет кашемира — прямое преимущество
  • Slow fashion / осознанная мода — растущий тренд
  • Российские бренды одежды — WS 3 500+ запросов
Threats — Угрозы
  • Маркетплейсы (WB, OZON) занимают топ по generic fashion-запросам
  • uniquefabric.ru активно наращивает контент (951 страница)
  • Если Яндекс не индексирует — технические причины могут быть сложнее
  • Тяжёлые страницы (Bitrix + слайдеры) сложно оптимизировать
  • Сезонность: /summer-2025/ конфликт URL при смене коллекций
