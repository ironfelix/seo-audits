02 · Технический SEO

Критические технические проблемы

Анализ HTTP-заголовков, HTML структуры, robots.txt, sitemap, meta-тегов. Платформа: Next.js App Router, поддомен shop.jevou.ru.

P0
Только 1 страница в индексе из 553+ блокирующее
Keys.so показывает pagesinindex=1 — только главная jevou.ru/. Весь каталог, категории, карточки товаров — невидимы поисковикам с июля 2025 (судя по истории). Потенциальные тысячи визитов в месяц не реализуются.
Источник: keys.so domain_dashboard · История: стабильно =1 с 2025.07
P0
Нет robots.txt — возвращает HTML страницы блокирующее
Запрос shop.jevou.ru/robots.txt → HTTP 200 с content-type: text/html. Реального файла robots.txt нет. Next.js рендерит страницу приложения вместо текстового файла. Поисковые боты не получают инструкций по обходу.
Проверено: curl -I shop.jevou.ru/robots.txt → 200 text/html
P0
Нет sitemap.xml — возвращает HTML страницы блокирующее
shop.jevou.ru/sitemap.xml возвращает HTTP 200 с полным HTML приложения вместо XML-карты сайта. Поисковики не знают о существовании 500+ страниц каталога.
Проверено: curl shop.jevou.ru/sitemap.xml → Content-Type: text/html
P1
Cache-Control: no-store на всех страницах высокий
Все ответы сервера содержат: cache-control: private, no-cache, no-store, max-age=0, must-revalidate. Страницы не кешируются CDN, каждый визит — запрос к серверу. Для SEO: боты видят нестабильный контент, нет cache-hit.
Затронуты: все страницы включая /robots.txt, /sitemap.xml
P1
OG-изображения ведут на внутренний IP высокий
В meta-тегах: og:image: http://111.88.147.179:8000/opengraph-image.jpg. При шаринге ВКонтакте, Telegram, WhatsApp — изображение не загружается. Социальный трафик теряет превью.
Влияет: все соцсети с OG-парсингом
P1
Redirect loop: / → /ru → / высокий
Корень сайта shop.jevou.ru/ возвращает 307 → /ru, который возвращает 3 байта (снова редирект). Curl с флагом --max-redirs завершает запрос с ошибкой "too many redirects". Ряд поисковых ботов не проходит эту цепочку.
Проверено: curl -v shop.jevou.ru/ → 307 /ru → 3 bytes
P2
Нет schema.org разметки среднее
На главной и во всём доступном HTML — ни одного блока application/ld+json. Отсутствуют: Organization, Product, BreadcrumbList, ItemList. Нет расширенных сниппетов, нет цен в поиске, нет хлебных крошек.
P2
Нет canonical и hreflang тегов среднее
Несмотря на country-code routing (/ru/), нет ни canonical, ни hreflang. При наличии /ru/, /en/ и других локалей — поисковик не понимает, какая страница приоритетна. Канонизация фильтрации (/ru/catalog?sort=price) также отсутствует.
P2
Магазин на поддомене shop.jevou.ru среднее
Link equity и DR (6) накапливаются на поддомен, а не на корневой домен. При любых обратных ссылках на jevou.ru (без shop.) авторитет не передаётся магазину. Рекомендуется: jevou.ru/shop/ или перенос всего на jevou.ru.

03 · Контент и семантика

Текущая семантика и контентные пробелы

Сайт не присутствует ни по одному коммерческому запросу. Потенциальный трафик по 553 SKU — полностью недополучается.

Главный инсайт по контенту: Две позиции в топ-50 — брендовый запрос с опечаткой («jevi ru», pos.26) и навигационный («бижутерия гринвич», pos.27, ws 39). Коммерческих запросов типа «серьги купить», «купить бижутерию онлайн» — ноль. Это не семантическая проблема, это следствие технической блокировки.

Контентные пробелы

❌
Нет SEO-текстов на категорийных страницах
Страницы /ru/catalog/earrings, /rings, /necklaces — пустые листинги без H1, описания, внутренних ссылок. По стандартам Яндекса категорийная страница должна иметь ~200–500 слов текста с ключевыми запросами кластера.
⚠️
Title главной не оптимизирован под ключевые запросы
Текущий title: «JEVǑU — трендовая бижутерия · JEVǑU» — бренд повторяется дважды, нет целевого запроса. Рекомендуется: «Бижутерия JEVǑU — серьги, кольца, колье купить с доставкой по России».
⚠️
Отсутствует контент под подарочный спрос (10K+ WS/мес)
Запросы «украшения в подарок девушке», «украшения подарок на день рождения» — суммарно 10 000+ запросов в месяц. Нет лендинга, нет категории «Подарки», нет фильтрации по бюджету.
💡
Блог существует, но контент неизвестен
В навигации есть ссылка «Блог». Информационный трафик (how-to, тренды сезона, гиды по подбору украшений) — важный канал для ЦА 16-30. Необходима проверка и наполнение.

06 · Quick Wins

Быстрые победы (первые 4 недели)

Действия, которые дают результат без масштабного рефакторинга. Фокус — разблокировать индексацию как можно скорее.

1. Создать robots.txt с явными директивами
Усилие: 1 час
Impact: Разблокирует краулинг
→ Размещение статического robots.txt в Next.js через app/robots.ts или public/robots.txt
2. Создать sitemap.xml со всеми категориями и товарами
Усилие: 4–8 часов (разработка)
Impact: 553+ страниц попадают в очередь индексации
→ Next.js: app/sitemap.ts или статический XML, отправить в Яндекс.Вебмастер и GSC
3. Исправить OG-изображение: убрать внутренний IP
Усилие: 30 минут
Impact: Соцсети начнут показывать превью
→ В opengraph-image.tsx или next.config.js заменить baseUrl на https://shop.jevou.ru
4. Убрать Cache-Control: no-store с публичных страниц
Усилие: 1–2 часа
Impact: CDN-кеш, LCP улучшится, ботам легче краулить
→ В Next.js: export const revalidate = 3600 или через next.config headers
5. Оптимизировать Title и H1 главной страницы
Усилие: 1 час
Impact: CTR из поиска +20–40%
→ Title: «Бижутерия JEVǑU — серьги, кольца, колье с доставкой по России» · H1: «Трендовая бижутерия — украшения для тех, кто знает тренды»
6. Добавить LocalBusiness schema для офлайн точек
Усилие: 2 часа
Impact: Геолокальные запросы (Уфа, Екб, Воронеж)
→ JSON-LD с адресами, телефонами, часами работы на страницах магазинов
07 · Роадмап

План работ на 4 месяца

Приоритет: сначала разблокировать индексацию, потом масштабировать контент.

Неделя 1–2 Техническое восстановление
  • Создать robots.txt через Next.js app/robots.ts (Allow: /, Disallow: /api/)
  • Создать sitemap.xml: главная + 5 категорий + все ~553 карточки товаров
  • Исправить OG-изображение: убрать IP 111.88.147.179, использовать https://shop.jevou.ru/og.jpg
  • Убрать Cache-Control: no-store с каталогных и продуктовых страниц
  • Исправить redirect loop: /ru должен отдавать 200, а не redirect
  • Отправить сайт на переобход в Яндекс.Вебмастер и Google Search Console
Неделя 3–4 Базовая on-page оптимизация
  • Добавить canonical теги на все страницы (включая фильтрацию ?sort=)
  • Добавить hreflang: ru-RU → /ru/, и x-default
  • Добавить schema.org: Organization (главная), Product (карточки), BreadcrumbList, ItemList (категории)
  • Оптимизировать Title/H1 для 5 категорийных страниц под ключевые кластеры
  • Добавить 150–300 слов описания на каждую категорийную страницу
Месяц 2 Контент-стратегия и подарочный кластер
  • Запустить подарочный лендинг: «Украшения в подарок» с фильтрацией по бюджету (до 2000₽, до 3000₽, до 5000₽)
  • Написать 4–6 статей в блог под информационный спрос ЦА: «Трендовые серьги сезона», «Как выбрать украшения под тип лица», «Гипоаллергенные украшения — что это»
  • SEO-оптимизировать Title/Description для топ-100 SKU по частотности запросов
  • Добавить страницы офлайн магазинов с LocalBusiness schema
Месяц 3–4 Рост ссылочного профиля и масштабирование
  • Получить 10–15 качественных обратных ссылок: lifestyle-блоги, обзоры, коллаборации с инфлюенсерами
  • Расширить контент-план до 2–3 статей/мес: тренды, гиды, сезонные подборки
  • Проработать фильтрацию: «серьги позолоченные», «серьги с жемчугом», «серьги-кольца» как отдельные SEO-страницы
  • Настроить аналитику: Яндекс.Метрика цели, GSC, отслеживание конверсий по категориям
  • Обновлять sitemap автоматически при добавлении новых товаров

09 · SWOT

SWOT-анализ

Позиция JEVOU относительно SEO-среды и рынка бижутерии.

Сильные стороны
  • Чёткое позиционирование «тренды + молодёжь 16-30» — дифференциатор от конкурентов
  • Широкий каталог: 553+ SKU в 5 категориях — много страниц под long-tail
  • Офлайн точки в 3 городах — возможность локального SEO
  • Next.js с SSR — техническая база для быстрого рендеринга
  • Гипоаллергенные сплавы — USP под отдельный поисковый кластер
  • Активный Instagram и социальное присутствие
Слабые стороны
  • 1 страница в индексе из 553+ — сайт органически не существует
  • DR=6 — минимальный авторитет домена, трудно конкурировать
  • Поддомен shop.jevou.ru — раздробленный link equity
  • Нет robots.txt, sitemap, schema.org — базовые пробелы
  • OG-изображения сломаны (внутренний IP)
  • Cache-Control: no-store — производительность и кеш
Возможности
  • Ниша «трендовая молодёжная бижутерия» — пустая в органике
  • Конкуренты-магазины (evabijou, evarits) — DR<5, можно обогнать быстро
  • Подарочный кластер 18K+ WS/мес — без конкурентов из нишевых магазинов
  • Информационный трафик (тренды, гиды) — не занят, высокий спрос у ЦА
  • Локальное SEO по 3 городам — Уфа, Екб, Воронеж практически пустые
  • Рост AI-трафика (Яндекс Нейро, ГигаЧат) — для молодёжных брендов
Угрозы
  • Wildberries и Ozon занимают топ-3 по категорийным транзакционным запросам
  • Медленный цикл индексации Яндексом новых страниц (2–8 недель)
  • Конкуренты могут также оптимизироваться и занять нишу первыми
  • Сезонность: бижутерия имеет пики (8 марта, НГ, Valentine's) — нужно успеть
  • Pinterest, Instagram — конкуренты за внимание ЦА в поисковике
